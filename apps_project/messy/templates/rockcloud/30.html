【预备知识】

防火墙(英文：firewall)是一项协助确保信息安全的设备，会依照特定的规则，允许或是限制传输的数据通过。防火墙可以是一台专属的硬件也可以是架设在一般硬件上的一套软件。Windows防火墙顾名思义就是在Windows操作系统中系统自带的软件防火墙。

防火墙对于每一个电脑用户的重要性不言而喻，尤其是在当前网络威胁泛滥的环境下，通过专业可靠的工具来帮助自己保护电脑信息安全十分重要。

【实验步骤】

网络拓扑：server2008-basic--win7

windows server 2008
用户：administrator
密码：Admin123

win7
用户：test
密码：123456

第一步：点击启动选项，启动实验虚拟机。

第二步：打开虚拟机Windows server 2008 R2，进入到系统桌面。

第三步：在任务栏中点击“开始”，点击“控制面板”。

第四步：在控制面板中点击“系统安全”找到Windows防火墙。

第五步：在系统和安全中点击“Windows防火墙”。

第六步：在windows防火墙界面中点击“高级设置”。（将防火墙开启）

第七步：在高级安全windows防火墙中选择入站规则点击“新建规则”。

第八步：进入规则向导页面，选择“端口”类型。

第九步：规则应用于TCP，选择“所有端口”

第十步：然后选择“允许链接”。

第十一步：默认选项，点击“下一步”。

第十二步：填写规则名称，如test。点击“完成”。

第十三步：在入站规则中双击新建的test规则，选择协议和端口，将协议类型改为ICMPv4。

第十四步：在高级安全windows防火墙中选择出站规则点击“新建规则”。

第十五步：进入规则向导页面，选择“端口”类型

第十六步：规则应用于TCP，选择“所有端口”

第十七步：然后选择“阻止链接”。

第十八步：默认选项，点击“下一步”。

第十九步：填写规则名称，如test。点击“完成”。

第二十步：在出站规则中双击新建的test规则，选择协议和端口，将协议类型改为ICMPv4。

第二十一步：打开命令指示符，看查本机地址，在命令指示符中输入ipconfig

第二十二步：打开虚拟机windows7，在windows7中打开命令指示符并查看本地地址

第二十三步：用windows server2008对windows7进行ping

第二十四步：用windows7对windows server2008进行ping

 

 

实验结束，关闭虚拟机。
