【预备知识】

了解网络地址，并了解基础dos操作。

【实验步骤】

网络拓扑：server2008-backup

windows server 2008
用户：administrator
密码：Admin123

第一步：点击启动选项，启动实验虚拟机。

第二步：打开windows server 2008虚拟机，进入到系统桌面环境。

第三步：在键盘上点击“windows键”和“R键”，打开运行框，在搜索栏中输入“cmd”。打开命令指示符。

第四步：在命令指示符中输入“ipconfig /all >D:/bak.txt”将网络配置信息保存至本地磁盘D

第五步：打开本地磁盘D找到文件“bak.txt”双击文件，看查信息是否与命令指示符中的相同

 

 

实验结束，关闭虚拟机。
