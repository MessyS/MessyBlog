【背景描述】

XX企业，为加强服务器建设，搭建了企业内部服务器，用于自身网站的建设，小王是该企业新进服务器管理员，承担服务器的管理工作。

现该企业网络存在如下需求：

在企业中搭建的windows服务器，防火墙需要定期维护

【实训设备】

产品型号

产品描述

数量

PY-P8

实训室安全、应用环境实训平台

1

【预备知识】

防火墙(英文：firewall)是一项协助确保信息安全的设备，会依照特定的规则，允许或是限制传输的数据通过。防火墙可以是一台专属的硬件也可以是架设在一般硬件上的一套软件。Windows防火墙顾名思义就是在Windows操作系统中系统自带的软件防火墙。

防火墙对于每一个电脑用户的重要性不言而喻，尤其是在当前网络威胁泛滥的环境下，通过专业可靠的工具来帮助自己保护电脑信息安全十分重要。

【实验步骤】

网络拓扑：server2008-basic--win7

windows server 2008
用户：administrator
密码：Admin123

win7
用户：test
密码：123456

第一步：点击启动选项，启动实验虚拟机。

第二步：打开虚拟机Windows server 2008 R2，进入到系统桌面。

 第三步：在任务栏中点击“开始，”点击“控制面板”。

第四步：在控制面板中点击“系统和安全”。

第五步：在系统和安全中点击“Windows防火墙”。

第六步：在windows防火墙中点击“更改通知设置”。

第七步：Windows防火墙阻止新应用时通知我：得到Windows防火墙是否进行了阻止，然后单击确定。阻止所有传入连接，包括位于允许应用列表中的应用：杜绝被入侵的危害。

第八步：在任务栏中点击“开始”，找到管理工具，点击“internnet信息服务（IIS）管理器”

第九步：在internnet信息服务（IIS）管理器界面中点击网址，点击添加ftp站点，进行站点设置。

第十步：在添加ftp站点中，ftp名称为ftp，路径为D盘下的ftp文件夹（需要自行创建）。

第十一步：不适用SSL，点击下一步继续

第十二步：身份验证为匿名，允许匿名用户访问。

第十三步：在D盘创建一个test文件。

第十四步：打开命令指示符，查看本机地址，在命令指示符中输入ipconfig。

第十五步：启动虚拟机windows7，打开计算机，在路径栏中输入ftp服务器的地址。例如ftp：//172.16.1.28。

第十六步：在server2008basic中，点击开始，找到管理工具，点击高级安全windows防火墙。

第十七步：在高级安全windows防火墙中，点击入站规则，选择ftp服务（ftp流入量）。

第十八步：在操作栏中，将允许连接改为阻止连接，点击确定即可。

第十九步：查看规则是否关闭。

第二十步：选择监视，点击防火墙，双击FTP服务器（FTP流入量），看查服务详情。

第二十一步：再次使用windows7访问ftp服务器，发现无法访问。

 

 

实验结束，关闭虚拟机。
